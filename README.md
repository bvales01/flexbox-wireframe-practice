# FLEXBOX-WIREFRAME-PRACTICE

The goal and purpose of this project was to practice building a responsive layout using CSS Flexbox. 

Technologies Used: 

**HTML5** for semantic structure

**CSS3** for custom styling and representation

**Flexbox** to manage layout and alignment and boxes
